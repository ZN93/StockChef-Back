# 🧑‍🍳 StockChef-back

Backend de l’application StockChef développé avec **Spring Boot 3.2 (Java 21)**.

## 🚀 Démarrage local

```bash
./mvnw spring-boot:run
```

## 📁 Structure

- `src/main/java` : code principal
- `src/test/java` : tests unitaires
- `application.properties` : configuration DB

## 🧪 Tests

```bash
./mvnw test
```
